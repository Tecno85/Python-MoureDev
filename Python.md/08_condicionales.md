# `Condicionales`

Los condicionales en `Python` permiten ejecutar código dependiendo de si se cumple o no una condición. El más común es el condicional `if`, seguido de `elif `(else if) y `else` para manejar varias condiciones.

**_Estructura básica de los condicionales_**:

- **`if`**: Se usa para evaluar una condición. Si es verdadera, se ejecuta el bloque de código debajo del `if`.

- **`elif`**: Se usa para evaluar condiciones adicionales si el primer `if` es falso. Puedes tener múltiples `elif`.

- **`else`**: Se usa para ejecutar código si todas las condiciones anteriores son falsas.


> ### _Sintaxis Básica_

```py
if condición
    # código a ejecutar si la condición es verdadera

# Ejemplo:
edad = 18

if edad >= 18:
    print("Eres mayor de edad.")

# Imprime: Eres mayor de edad
```

En este ejemplo si la edad es mayor o  igual a 18, se imprime el mensaje `"Eres mayor de eadad."`
